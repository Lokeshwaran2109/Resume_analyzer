# 🚀 AI Resume Analyzer & Career Guide

An end-to-end AI-powered web application that analyzes resumes, compares them with job descriptions, and provides intelligent career insights using NLP and Generative AI.

---

## 🔍 Features

- 📄 Upload Resume (PDF)
- 🧠 Extract Skills using NLP
- 📊 Job Description Matching
- 🎯 Match Score Calculation
- ❌ Missing Skills Identification
- 🤖 AI Career Suggestions (Azure OpenAI)
- 🌐 Deployed on Microsoft Azure

---

## 🛠 Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, PyPDF2  
- **AI/NLP:** Skill Extraction + Azure OpenAI  
- **Cloud:** Microsoft Azure App Service  

---

## ⚙️ How It Works

1. Upload your resume (PDF)
2. Paste a job description
3. System extracts skills using NLP
4. Compares resume vs job description
5. Calculates match score
6. Uses AI to generate:
   - Career suggestions
   - Missing skills
   - Improvement tips
